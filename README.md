# CMU-Africa Deadline Tracker 🎓📅

A sleek, functional Android application built to help CMU-Africa students manage their rigorous academic schedule by tracking assignments, course tasks, and deadlines.

## ✨ Key Features
- **Dashboard Overview:** Quick-view cards showing "Remaining", "Completed", and "Due This Week" tasks.
- **Task Management:** Categorized views for Assignments, Courses, and Schedules.
- **Completion Tracking:** Mark tasks as "Done" to keep your list organized.
- **Clean UI:** Minimalist design with light/dark mode support (indicated by the sun icon).

## 🛠 Tech Stack

## 💡 Motivation
I created this app to solve the personal challenge of managing multiple deadlines at CMU-Africa, ensuring no homework or proposal falls through the cracks.
